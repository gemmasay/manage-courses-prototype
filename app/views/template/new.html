Use templates to copy information across courses.

You should create a separate template for each type of course you offer. Eg, one template for all your QTS salaried courses and one for all your PGCE with QTS courses.

Build a template by entering content in the free text fields. This content will appear in all the profiles that use this template – you’ll be able to apply it to all relevant courses through a clickable list.

Before you create your template, give it a name that describes the type of course it applies to.
